# DeepCSAT – Ecommerce Customer Satisfaction Prediction

This project analyzes ecommerce customer support data and predicts Customer Satisfaction Score (CSAT) using machine learning.

## Project Type
EDA + Classification

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Techniques Used
- Data Cleaning
- Feature Engineering
- Data Visualization
- Machine Learning
- Cross Validation
- Hyperparameter Tuning

## Tools & Libraries
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn

## Result
Random Forest achieved the best performance with approximately **65% accuracy**.
